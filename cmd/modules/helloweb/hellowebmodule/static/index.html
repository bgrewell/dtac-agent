DTAC Hello Web Module

This is an example web module for the DTAC framework.

Module Type: Web Module with Embedded Static Assets
Technology: Go + fs.embed
Purpose: Demonstrate the DTAC module system for hosting web frontends

Features

  • Embedded static assets (no external file dependencies)
  • Integrated with DTAC module system
  • Structured logging to DTAC agent
  • Extensible for proxy routes and authentication

This module runs as a separate process managed by the DTAC agent.
